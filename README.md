# 📊 Next-Gen AI Analytics Dashboard for Adaptive Learning  

A **web-based AI-powered analytics system** that transforms academic data into actionable insights and personalized study plans.

This project demonstrates how intelligent analysis and adaptive logic can improve student performance beyond just marks.

---

# 🔗 Live Demo  

👉 https://09amulya.github.io/Next-Gen-AI-Analytics-Dashboard-for-Adaptive-Learning/

---

# 📌 Features  

## 🔍 Performance Analysis  

- Analyze student marks across subjects  
- Identify strengths and weaknesses  
- Detect performance trends over time  

## 📊 Interactive Dashboard  

- Graphical representation of performance  
- Subject-wise comparison  
- Visual charts with color indicators  

## 📅 Adaptive Study Plan  

- Automatically generated study schedules  
- Focus based on weak areas  
- Dynamic adjustment with new data  

## 🤖 AI-Based Insights  

- Smart recommendations for improvement  
- Performance-based categorization  
- Continuous learning adaptation  

---

# 🧠 Adaptive Learning Logic  

## 📊 Performance Levels  

| Category | Percentage |
|----------|-----------|
| 🔴 Weak  | < 50%     |
| 🟡 Medium | 50–75%   |
| 🟢 Strong | > 75%    |

---

## 📅 Study Strategy  

| Level   | Time Allocation | Strategy              |
|--------|---------------|----------------------|
| Weak   | High          | Revise + Practice    |
| Medium | Moderate      | Practice + Improve   |
| Strong | Low           | Periodic Revision    |

---

## 🔄 Dynamic Adaptation  

- Improvement → Reduced focus  
- Decline → Increased focus

# ⚙️ Tech Stack  

- **Frontend:** HTML, CSS, JavaScript / React  
- **Backend:** Python (Flask) / Node.js  
- **Visualization:** Chart.js  
- **Database:** JSON / MongoDB  
- **PDF Processing:** PDF → JSON Conversion  
- **OCR (Optional):** For scanned PDFs  

---

# 💻 Core Modules  

- **PDF Parser** → Extracts marks data  
- **Data Structuring** → Converts into JSON  
- **Analysis Engine** → Detects patterns  
- **Plan Generator** → Creates study schedules

# 🎯 Project Objective

# ⚠️ Limitations

- OCR may introduce minor errors  
- Only marks-based analysis

- Time constraints
